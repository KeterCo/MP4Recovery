批量修复视频真实时长工具，选择个文件夹，有预览待处理视频列表的窗口界面。可自定义文件名后缀、同步修改同名.ass字幕文件和.xml配置文件。可选择是否递归处理文件夹下所有的视频文件。
python3.12.3,使用ffmpeg工具，目前仅支持mp4，win10可用。
修改：原视频大于100MB的视频增加了个判断逻辑，若生成的视频与原视频差距大于25%，则不处理该视频。
本地程序配置文件目录在：C:\Users\xxx\AppData\Local\Mp4recovery
